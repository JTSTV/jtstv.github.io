bash: hexo: command not found root@localhost:~/hexo#

根据您我搜索到的资料，bash: hexo: command not found 错误通常是由于 Hexo 没有正确安装或其路径未添加到系统的环境变量中。以下是详细的解决步骤：

全局安装 Hexo：
首先，确保您已经全局安装了 Hexo CLI。可以使用以下命令进行安装：

npm install -g hexo-cli
检查 Node.js 和 npm 版本：
确保您的 Node.js 和 npm 版本符合要求。根据证据，Node.js 版本应高于 8.6.0，npm 版本应为最新版本。您可以使用以下命令检查版本：

node -v
npm -v
检查 Hexo 是否安装成功：
安装完成后，可以通过以下命令检查 Hexo 是否安装成功：

hexo –version
如果仍然出现 bash: hexo: command not found 错误，则需要进一步检查环境变量配置。# 编辑保存退出
